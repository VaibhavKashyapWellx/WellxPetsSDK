# WellX Pets — Custom Fonts

## Inter Font Family

Download Inter from https://rsms.me/inter/ (SIL Open Font License 1.1)

Required files:
- `Inter-Regular.ttf` (weight 400)
- `Inter-Medium.ttf` (weight 500)
- `Inter-SemiBold.ttf` (weight 600)
- `Inter-Bold.ttf` (weight 700)

These files are excluded from version control (.gitignore) because they are
large binary files. Download and place them in this directory before building.

You can also use the `google_fonts` package as an alternative:
```dart
import 'package:google_fonts/google_fonts.dart';
// GoogleFonts.inter(fontSize: 16)
```
